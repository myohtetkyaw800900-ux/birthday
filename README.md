# 🎂 Interactive 3D Birthday Project

Vite Build Tool ကို အခြေခံပြီး **Three.js**, **HTML5**, **CSS3**, နှင့် **Vanilla JavaScript** တို့ကို ပေါင်းစပ်ဖန်တီးထားသော မွေးနေ့ဆုတောင်း Project တစ်ခု ဖြစ်သည်။

---

## 🌟 Features

- **3D Rendering:** Three.js Library ကို အသုံးပြု၍ လှပသော 3D scene များ ဖန်တီးထားခြင်း။
- **Fast Build:** Vite ကို အသုံးပြုထားသဖြင့် Development လုပ်ဆောင်ချက် မြန်ဆန်ခြင်း။
- **Responsive Layout:** Mobile နှင့် Desktop device များအားလုံးတွင် ကြည့်ရှုနိုင်ရန် ပြုလုပ်ထားခြင်း။
- **Interactive Elements:** အသုံးပြုသူမှ 3D objects များနှင့် interaction လုပ်နိုင်ခြင်း။

---

## 🛠️ Tech Stack

- **Vite** - Next Generation Frontend Tooling
- **Three.js** - 3D JavaScript Library
- **HTML5 / CSS3** - Core Web Structure & Styling
- **JavaScript (ES6+)** - Logic & Animation Control

---

## 🚀 Getting Started

ဒီ Project ကို ကိုယ့်စက်ထဲမှာ Run ဖို့အတွက် အောက်ပါ command များကို အသုံးပြုပါ။

### 1. Project ကို Clone လုပ်ပါ
```bash
git clone [https://github.com/kyawzin17/Birthday.git](https://github.com/kyawzin17/Birthday.git)
cd Birthday
```  
### 2. လိုအပ်သော packages များ install လုပ်ပါ
```bash
npm install
```

### 3. Development Mode မှာ Run လုပ်ပါ
```bash
npm run dev
```
Project ကို အောက်ပါ link မှ ကြည့်ရှုနိုင်သည်။
```
http://localhost:5173
```
